# AGENTS.md

This file provides instructions for AI agents working on this project.

## Getting Started

To get started with this project, clone the repository and install the dependencies.

```bash
git clone <repository-url>
cd obsidian-templater-core-kbt
```

There are no dependencies to install for this project.

## Running Tests

To run the tests, use the following command:

```bash
npm test
```

Currently, there are no tests for this project.

## Code Style

This project follows the standard JavaScript conventions. Please adhere to the following guidelines:

- Use Prettier for code formatting.
- Use camelCase for variable and function names.
- Use PascalCase for class names.
- Add comments to explain complex logic.

## Project Structure

The project has the following structure:

- `src/`: Contains the source code for the project.
  - `index.js`: The main entry point of the application.
  - `custom.js`: Contains custom scripts.
- `templates/`: Contains templates for generating notes.
- `package.json`: Contains project metadata and dependencies.

## Pull Request Guidelines

Before submitting a pull request, please ensure the following:

- Your code builds successfully.
- You have added or updated tests for your changes.
- Your code follows the project's code style guidelines.
- Your commit messages are clear and descriptive.
